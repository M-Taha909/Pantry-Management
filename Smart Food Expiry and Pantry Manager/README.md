# PantryPal 🥗

A full-stack MERN application that helps users manage pantry items, track food consumption, monitor expiry dates, analyze food waste, and reduce unnecessary grocery expenses.

---

## Features

- User Authentication (JWT)
- Pantry Management (CRUD)
- Food Consumption Tracking
- Remaining Quantity Monitoring
- Expiry Date Tracking
- Food Health Score
- Financial Analytics
- Category Statistics
- Smart Food Recommendations
- Mobile Responsive Design
- Toast Notifications
- Secure REST APIs

---

## Tech Stack

### Frontend

- React

- Axios

### Backend

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/PantryPal.git

cd PantryPal
```

### Backend

```bash
cd backend

npm install

npm run dev
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_uri

JWT_SECRET=your_secret

CLIENT_URL=http://localhost:5173
```
